Conceitos de POO Aplicados
Encapsulamento
Dados encapsulados com getters e setters, controlando acesso aos atributos.
Exemplo:
class Product { private BigDecimal price; public getPrice() { } }
Herança
Classes herdam funcionalidades comuns, reduzindo duplicação de código.
Exemplo:
JpaRepository ↓ ProductRepository (herda CRUD)
Polimorfismo
Tipos de movimentação (ENTRY/EXIT) com comportamentos diferentes.
Exemplo:
enum MovementType { ENTRY, EXIT }
Abstração
Interfaces definem contratos, permitindo múltiplas implementações.
Exemplo:
interface Repository { save(T entity); findById(id); }
Composição
StockMovement composto por referências a Product e User.
Exemplo:
class StockMovement { Product product; User user; }
