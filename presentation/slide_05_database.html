Modelo de Dados
User (Usuários)
Representa os usuários do sistema com autenticação e controle de acesso.
id: Long (PK)
username: String
email: String
role: ADMIN | USER
Product (Produtos)
Representa os produtos no estoque com informações comerciais.
id: Long (PK)
name: String
price: BigDecimal
quantity: Integer
minQuantity: Integer
StockMovement (Movimentações)
Registra operações de entrada e saída de estoque.
id: Long (PK)
type: ENTRY | EXIT
quantity: Integer
createdAt: LocalDateTime
Relacionamentos
1 User → N Products → N StockMovements
